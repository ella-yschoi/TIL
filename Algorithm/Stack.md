# Stack

## 1. Stack의 구조

- 입력과 출력이 하나의 방향, 즉 스택의 최상단에서만 이루어지는 제한적 접근
- 이러한 Stack의 자료구조 정책: LIFO(Last In First Out) 혹은 FILO(First In Last Out)
- Stack에 데이터를 넣는 것: push, 꺼내는 것: pop

<br/>

## 2. Stack의 특징

- **LIFO(Last In First Out)**: 후입선출
- **하나의 입출력 방향**을 가짐
- 데이터는 **하나씩** 넣고 뺄 수 있음

<br/>

## 3. Stack 실사용 예제

- 브라우저 뒤로가기, 앞으로 가기 기능 구현
